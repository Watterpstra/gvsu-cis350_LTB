# Prerequisites
For the engine: 
1. download the Unity Game Editor https://unity.com/products/unity-engine
2. install and make sure it's up to date
   
# Run Instuctions:

Download and Unzip the files located in the Final_Project Folder
Open the Unity game engine hub -> projects -> add -> add from disk
The folder should show up as a viable project that Unity can run
load up the file and the first level should appear and be runable using the play, pause, and stop buttons at the top center of the UI.
